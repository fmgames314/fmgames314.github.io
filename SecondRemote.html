Second Remote


This was a simple controller created to be used when the main global controller is already in use. This allows another person to drive another robot at the same time. This controller has 2 joysticks a button and 3 potentiometers. So the controls are a little different than the main controller. Each robot has to allow this controller in its code. This controller was created from an old Cisco router, hence the poor cutting job
